# Avadon

Jeu 2D tournant sous pygame. Projet pour apprendre à coder un jeu, en suivant la vidéo https://www.youtube.com/watch?v=fo4e3njyGy0&t=708s de ClearCode. L'histoire et les mécaniques vont être différentes. Les cartes sont créées grâce à des sprites sheets et faites sur Tiled.
